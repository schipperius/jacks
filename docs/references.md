To create accurate citations, references, and bibliographies, you need to collect specific metadata that allows for the identification and retrieval of your sources. While specific styles (APA, MLA, Chicago) differ in formatting, the core metadata required is generally consistent: author, date, title, and source information. 

Here is a breakdown of the necessary metadata for different types of sources:

book: 
  author(s): full names
  (s)/Editor(s): Full names.
  Title of Book: Full title, including subtitle.
  Publication Date: Year of publication.
  Publisher: Name of the publishing house.
  Edition: Necessary if it is not the first edition.
  Volume/Series: If applicable.
  Place of Publication: City (often required in Chicago/older styles).
  ISBN: Optional, but useful for identification. 

journal_articles: 
  Author(s): Individual or group authors.
  Article Title: Title of the specific article.
  Journal Title: Name of the journal.
  Date: Year (and month/day if applicable).
  Volume and Issue Number:.
  Page Range: The first and last page of the article.
  DOI or URL: Digital Object Identifier (preferred) or direct link. 

website_online_content:
  Author(s): Personal or organizational author.
  Title of Page/Article:.
  Website Name/Publisher:.
  Publication Date: Or "n.d." if no date is available.
  URL:.
  Access Date: Required if the publication date is unknown or if content is likely to change. 

4. Other Specialized Sources
Datasets: Creator, date, title, version, publisher, DOI/persistent identifier.
Films/Videos: Director, title, studio, release year, medium (DVD, streaming).
Sound Recordings: Composer/performer, song title, album, label, year, medium.
Book Chapters (Edited Volumes): Chapter author, chapter title, book editor, book title, page range, publisher, year. 

Key Differences: References vs. Notes
References/Bibliography: These appear at the end of the paper and require full, detailed information (e.g., author, full title, publisher, date, URL/DOI).
Notes (Footnotes/Endnotes): Often used in Chicago style. The first note for a source requires the full citation. Subsequent notes for the same source can use a shortened format: Author Last Name, Shortened Title, and page number. 

Pro-Tips for Metadata Collection
Use Tools: Use reference managers like Zotero to save this metadata automatically from browsers.
Prioritize Stability: Use DOIs for journal articles rather than URLs, as they are permanent.
Be Consistent: Whichever style you use, apply it uniformly to all entries.
Check for "No Author": If no author is listed, use the organization name or the title of the work.
